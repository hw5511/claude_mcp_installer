# HTML to Notion MCP

Claude에서 생성된 HTML 코드를 GitHub Gist에 업로드하고 Notion에 쉽게 임베드할 수 있게 해주는 MCP입니다.

## 기능

- Claude가 생성한 HTML 코드를 GitHub Gist로 변환
- Notion에서 임베드 가능한 URL 생성
- HTML 유효성 검사 및 최적화

## 설치 요구사항

- GitHub 계정 및 Personal Access Token (gist 권한 필요)
- Python 3.6 이상
- `requests` 및 `fastmcp` Python 라이브러리 (자동 설치됨)

## 사용 방법

1. Claude에게 HTML 코드 생성을 요청합니다.
2. 생성된 코드를 Notion에 임베드하고 싶다고 요청합니다.
3. Claude가 MCP를 통해 GitHub Gist URL을 생성합니다.
4. 제공된 URL을 Notion의 `/embed` 블록에 붙여넣습니다.

## API 사용법

### HTML을 Notion 임베드 URL로 변환

**엔드포인트**: `/html-to-notion`
**메서드**: `POST`
**요청 본문**:

```json
{
  "html": "<html>...</html>",
  "description": "HTML generated by Claude"
}
```

**응답**:

```json
{
  "success": true,
  "gist_id": "abcd1234",
  "raw_url": "https://gist.githubusercontent.com/username/abcd1234/raw",
  "embeddable_url": "https://gist.github.com/username/abcd1234",
  "is_accessible": true,
  "notion_embed_instructions": "In Notion, create a new block and type '/embed', then paste this URL."
}
```

## 문제 해결

- **인증 오류**: GitHub 토큰이 올바르게 설정되었는지 확인하세요.
- **HTML 유효성 오류**: 생성된 HTML 코드에 오류가 있는 경우 Claude에게 수정을 요청하세요.
- **URL 접근 오류**: 생성된 Gist가 공개(public)로 설정되어 있는지 확인하세요.
